2.有附字體，可以的話安裝一下，不然有些東西會跑掉

3.有音效 請把聲音打開，開小聲一點

4.一開始和結束有小動畫

5.先看完說明才能玩遊戲，「看完」喔（字停止為止）

6.開始後或點技能後都會自動focus到輸入的textbox，可以直接打字

7.打完字之後按Enter就可以了（除了逗號之外都是半形，CTRL+ㄝ可以打逗號）

8.沒設難度，可是每過10秒字增加的頻率會增加（每10秒快200ms）

9.有技能，技能有CD喔，15秒

10.掉"?"下來而且輸入的話會有隨機事件（有4種）
